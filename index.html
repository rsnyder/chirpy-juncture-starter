Welcome!

This website template is built on Jekyll, a publishing system that turns simple text files into a complete website. It offers a practical way to create a modern site without needing to learn traditional web development. You write in clean, readable Markdown, much like writing an email or document, and the framework takes care of layout, navigation, styling, and publishing. With built-in support for images, video, diagrams, math, and structured formatting, you can create content that looks polished and professional from the start.

It builds on the popular Chirpy Jekyll theme, a clean and modern design with built-in navigation, search, and tagging. On top of that, it adds interactive features such as rich image viewers and entity information popups, allowing your pages to go beyond static text while remaining simple to author. The result is a lightweight but capable system that lets you focus on ideas and storytelling instead of technical setup. If you can edit text in a browser, you can build and maintain an engaging website with this framework.

The source code for this template is available on GitHub. You are welcome to explore it, adapt it, and use it as a starting point for your own projects. If you need help getting started, just reach out.
